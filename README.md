# Asset Tracking System

Ett enkelt konsolprojekt i C# för att hantera och visa tillgångar.

## Funktioner
- Lägg till tillgångar (typ, märke, modell, kontor, pris, inköpsdatum)
- Visar alla sparade tillgångar
- Konverterar pris från SEK till USD med live valutakurser från ECB

## Valutakonvertering
Systemet hämtar aktuella valutakurser från Europeiska Centralbanken och konverterar via EUR som mellanvaluta.

## Tekniker
- C#
- .NET Console App
- XML parsing
- Live API (ECB valutakurser)
